Klasifikasi Sampah

This project is a website aimed at applying the Support Vector Machine (SVM) algorithm in waste classification. For example, when a user enters a picture of a product such as toothpaste, the SVM algorithm will automatically classify it as toothpaste waste. The same will happen when entering a picture of soap or shampoo. This project is designed using Python for SVM algorithm implementation, and Flask is used as the framework for deployment. The user interface is presented with a responsive display based on HTML and CSS. By utilizing the sophistication of SVM algorithm, this project enables identification and classification of waste with high accuracy. Users can easily upload product images and get fast and effective classification results. The display components designed using HTML and CSS provide a clean and accessible interface, ensuring a good user experience.
